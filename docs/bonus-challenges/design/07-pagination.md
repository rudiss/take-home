# Bonus 7: Add Pagination or Infinite Scroll ⭐

**Difficulty:** Easy-Medium

---

## Goal

Handle large lists gracefully with pagination or infinite scroll.

---

## Option 1: Pagination

### UI Components

- Page number buttons
- Previous/Next buttons
- "Showing X-Y of Z results"
- Jump to page input

### Design Considerations

- Clear current page indicator
- Disable buttons appropriately
- Mobile-friendly sizing

---

## Option 2: Infinite Scroll

### Implementation

- Load more as user scrolls
- "Load More" button at bottom
- Loading indicator while fetching
- Smooth content insertion

### UX Tips

- Show loading state
- Handle loading errors
- Consider performance with many items

---

## What We're Looking For

Candidates who:

- Think about scalability
- Create smooth user experiences
- Handle loading and error states
- Consider performance implications
